Put a Wowhead search box in your sidebar. Has configurable sizes and content.

== Description ==

This plugin provides a sidebar widget containing a configurable Wowhead search box.

You can choose from the six search box options shown [here](http://www.wowhead.com/?spread).

== Installation ==

1. Upload `wowhead-search.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag Wowhead Search into your sidebar on the Appearance -> Widgets page.
